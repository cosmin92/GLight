Interactive graphics(final project) examples

User commands:

  • Mouse click & drag: rotate the viewer position around the (0,0,0) point in global coordinate system
  • Shift key + Mouse click & drag: translate the viewer position
  • Alt key + Mouse click: select an object in the scene
  • The following commands can be used to render a clean scene
    • Key1: show/hide x axis
    • Key2: show/hide y axis
    • Key3: show/hide z axis
    • Key4: show/hide axis
    • Key5: show/hide grid
    • Key6: show/hide lights and cameras()
  • KeyP + [KeyX | KeyY | KeyZ] + [ArrowUp | ArrowDown]: translate the pivot position
  • KeyT + [KeyX | KeyY | KeyZ] + [ArrowUp | ArrowDown]: translate the selected mesh
  • KeyR + [KeyX | KeyY | KeyZ] + [ArrowUp | ArrowDown]: rotate the selected mesh
  • KeyS + [KeyX | KeyY | KeyZ] + [ArrowUp | ArrowDown]: scale the selected mesh

Examples:

  1. Simple scene
    A simple scene where you can see some of the features of the library. In particular, this scene was built to show how the entities in the scene are displayed. You can also see the application of lights and materials. Furthermore, the use of the commands and the selection method can be verified
  2. Textures
    Here we can see the application of various textures to various meshes. And the first simple animation
  3. Hierarchical models
    Here a man and a dog are created as two forms of herarchical models. The use of the pivot point needs to be pointed out here. In fact, by selecting one of the components the transformations take place around the pivot and not outside the mesh (use the commands listed above).
  4. Easing
    Different types of easing and linear interpolation are applied to the spheres of this example
  5. Interpolations
    The library implements two types of interpolation: linear and BSpline. In the example to the same keyframes different types of interpolation are applied
  6. Complex animation
    Here you can see a short animation of a walking man using different types of easing and interpolation
